    Item

    Extends:

    Events → Model → ParseModel → Item

    Our basic todos Item model has content, order, and done attributes. When creating a new Item instance in App.js->createItem the current Parse user is also associated with the item instance along with a Parse.ACL (access control list) instance which ensures that the item is only accessible by that given user.
