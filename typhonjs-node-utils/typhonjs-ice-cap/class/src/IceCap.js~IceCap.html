    IceCap

    IceCap provides programmable HTML templating with cheerio.

    Example:

    import IceCap from 'typhonjs-ice-cap';

const ice = new IceCap('<p data-ice="name"></p>');
ice.text('name', 'Alice');
console.log(ice.html); // <p data-ice="name">Alice</p>
